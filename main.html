Recipe Picker

For my JavaScript class, I created this Recipe Picker. The purpose of this website is to help users figure out what to cook. The questionnaire asks the user about flavour, temperature and time and displays recipes depending on the choices you made. I used an inline frame to display the recipes. The tools I used were HTML, CSS and JavaScript.

Humber Centre of Innovative Learning

The Centre for Innovative Learning (CIL) is dedicated to helping Humber faculty members develop and enhance teaching practices.
